You are given a number N. Let us say the floor ( Ref :https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/floor) of 32 divided by N is x

Print as per following rules

* In case x is 0, print "Too Low"

* In case it is not possible to generate a valid number, print -1

* In all other cases, print x


Input
Input Format :

First line contains N


Constraints :

N <= 100


Output
Output string or number based on the conditions mentioned above


Sample Input 1 

4
Sample Output 1

8
